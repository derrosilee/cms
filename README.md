## Content management system for shops

### to be able to track current stock for a specific shop

<ol>
    <li>Shop registration</li>
    <li>Product registration</li>
    <li>Stock registration</li>
    <li>Stock update</li>
    <li>Stock report</li>
</ol>

### Shop registration
- Shop name
- Shop address
- Shop phone number
- Shop email
- Shop owner
- Shop registration date
- Shop status
- Shop Category
- Shop location
- Shop description

### Product registration
- Product name  
- Product description
- Product category
- Product price
- Product quantity
- Product registration date
